## 🔧 故障排除

### 常见问题及解决方案

1. **应用无法启动**
   ```bash
   # 检查端口是否被占用
   lsof -i :5001
   
   # 杀死占用进程
   pkill -f flask
   
   # 重新启动
   python app.py
   ```

2. **数据库错误**
   ```bash
   # 重置数据库
   rm instance/app.db
   python -c "from app import app, db; app.app_context().push(); db.create_all()"
   
   # 创建测试账户
   python -c "from app import app; app.app_context().push(); app.cli.commands['create-admin'].callback()"
   ```

3. **API调用失败**
   - 检查网络连接
   - 确认API密钥配置
   - 查看控制台错误信息
